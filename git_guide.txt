1) git init - Инициализировать Гит репозиторий в текущей папке
2) создаем .gitignore файл вносим в него все файлы, которых не хотим добавлять в гит
3) git status - Проверяем статус файлов в репозитории
4) git add . - Добавить все файлы в коммит
5) git commit -m 'Some Message' - Создаем коммит и подписываем его(Вместо Some Message)
6) git push - Отправяем файлы на гит сервер
Если это первый push этого репозитория, то вместое 6 пункта:
6.1) git remote add origin URL - Добавляем нашу ветку на сервер гита(Вместо URL ссылка на ваш гит репозиторий на GitHub)
6.2) git push --set-upstream origin master - Пушим и привязываем локальный репозиторий к репозиторию на сервере
